# Design Patterns in Javascript

This repository contains the tasks that are a part of the EPAM Design Paterrns in Javascript program

# How to run the application

# Step 1:
- Run `npm install` command to install the dependencies mentioned in the `package.json` file

# Step 2:

- Run `npm run build` to bundle the source files

# Step 3:

- After completing the above step, open `index.html` file to see the outputs of the individual tasks mentioned in the homework
